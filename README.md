# 🧠 行吗 (XingMa) - AI 法律风险评估 Skill

> 「先问行不行，再问好不好。」—— 风险在前，收益在后。

## 📖 项目简介
「行吗」是一个专为 AI Agent 设计的综合合规评估 Skill。输入任何商业计划、行为模式、合同条款或个人疑问，即可自动进行多维度合规扫描、风险等级评定与优化建议。

## 🛠️ 核心能力
- 🔍 **六维合规扫描**：政策法规 / 契约治理 / 权益伦理 / 财税资金 / 知产品牌 / 平台数据
- 🚦 **红绿灯评级系统**：🟢可行 → 🟡留意 → 🟠审慎 → 🔴不可行
- 🚩 **红旗法则**：8 条快速触发规则，触及即提高风险等级
- 📋 **文件审查支持**：支持逐条合同/条款审查与修改建议
- 📅 **2026 新规雷达**：涵盖延迟退休、新增值税法、新公司法等最新法规

## 📚 知识库信息源
本 Skill 的评估逻辑基于中华人民共和国现行公开法律法规（截至 2026 年 4 月）：
- 《中华人民共和国民法典》（合同编/人格权编/侵权责任编）
- 《中华人民共和国公司法》（2024 修订版）
- 《中华人民共和国增值税法》（2026 施行）
- 《个人信息保护法》/《数据安全法》/《网络安全法》
- 人社部关于渐进式延迟法定退休年龄的决定
- 工信部、网信办等部门规章及司法解释

## 🚀 使用方法
1. 将本目录放入你的 Agent 的 `skills/` 或 `active_skills/` 路径
2. 触发词：`行吗` / `这个合规吗` / `法律风险评估` / `帮我看看这个合同`
3. 输入你的计划/行为/合同内容，等待六维合规扫描报告
